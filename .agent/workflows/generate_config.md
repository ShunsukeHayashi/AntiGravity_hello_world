# Antigravity 設定の自動生成

1. ユーザーのリクエストを分析し、**ルール**（受動的な指針）か **ワークフロー**（能動的な手順）かを判断する。
2. コンテキストを理解するためにマスターガイドを読み込む。
// turbo
3. マスターガイドを表示: `view_file .agent/rules/antigravity_config_master.md`
4. マスターガイドの基準に従い、新しいファイルの内容を作成する。
5. 適切なディレクトリ（`.agent/rules/` または `.agent/workflows/`）にファイルを作成する。
6. 作成したファイルの場所をユーザーに通知する。
